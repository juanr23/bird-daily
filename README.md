# Bird of the Day

A simple website that shows a **daily bird fun fact** and photo. The same bird appears all day for everyone, then changes at midnight.

## Features

- Picks one of 11,000+ bird species based on today's date
- Shows a fun fact, photo, description, and quick stats
- "Surprise me" button for a random bird
- Data and images from [BirdFYI](https://birdfyi.com) and Wikimedia Commons

## Run locally

```bash
cd bird-daily
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

You can also open `index.html` directly in a browser, but a local server is recommended.
